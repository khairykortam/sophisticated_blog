# Sophisticated-blog

- The project is a simple personal website made using svelte.js and sveltekit framework.
- The project made me learn svelte basics such as props, components, and some other things.

#project structure

the project contains 5 routes, located in the routes folder, which are
```
\ -> home
\blog -> blogs
\book -> books
\entry1 -> blog entry 1
\book1 -> book entry 1
```
- assets are in 
```
static/assets
```
the resued components are the following
```
<navBar/> -> the bottom navigation bar
<main/> -> the main home page content
```
- The website features a secret, picture, it's you're role to find it!(hint: mouse hover)
